# TO DO APPLICATION

This app allows user to track the to dos.

### Instructions
1. Download the source code

```js
git clone [url to the github repository]
```
or just download the .zip file

2. Install dependencies
```js
npm install
```

3. Run the application
```js
node app
```

#### Dependencies list
- express
- pug
